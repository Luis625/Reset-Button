Descrição do projeto:

Foi projetado, montado e instalado um circuito que executa a seguinte lógica:

	- Recebe o sinal do botão instalado no local;
	- Aciona o relé, informando ao sistema que o mesmo deve ser reiniciado;
	- Liga a lâmpada do painel, indicando que o sistema foi reiniciado;
	- Trava o painel, impedindo que o sistema seja reiniciado por 5 minutos;
	- Após os 5 minutos, destrava o painel, permitindo a reinicialização do
sistema novamente e a lâmpada é desligada, informando que está liberada outra
reinicialização.
